# Create and Post Content

## What you can post

- Image post
- Video post (with thumbnail)
- Caption with hashtags (`#tag`) and token tags (`$token`)

## Posting checklist

1. Upload media.
2. Add caption.
3. Select chain/token.
4. Set base price.
5. Publish.

## Base price rules

- You must set a base price.
- Minimum value is enforced at about **$0.50 equivalent** in selected token.

## Upload limits

Hype enforces posting progression rules:

- Standard post threshold: 50
- Video post threshold: 20
- To keep posting beyond thresholds, you must own enough content from others.

This is designed to balance creating and collecting inside the app.

## Common posting errors

- Base price missing
- Price below minimum
- Video uploaded without thumbnail
- Both image and video uploaded at the same time
