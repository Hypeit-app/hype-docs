# Supported Networks

Hype supports multiple blockchain networks for posting, collecting, and marketplace actions.

## What you can do

- Choose the network that fits your wallet and fees
- Understand basic gas behavior
- Confirm wallet compatibility before transactions

## Networks

Hype supports Base, Ethereum, and Shibarium. For most users, Base is the default and recommended network.

If you are starting fresh, use Base unless you have a specific reason to use another chain.

## How to choose a network

1. Check which network your wallet already uses.
2. Compare fee levels before posting or collecting.
3. If unsure, choose Base.

## Network priority

- Primary focus: Base
- Also supported: Ethereum
- Legacy support: Shibarium

## Gas explanation (simple)

Gas is the network fee required for transaction actions. When you collect, list, or transfer, you pay gas on the selected network. The exact amount changes with network traffic.

## Wallet compatibility

Most EVM wallets are supported when configured for the selected network.

If a wallet is not configured for a network, add that network first in your wallet app.
